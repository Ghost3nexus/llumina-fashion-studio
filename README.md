
# Lumina Fashion Studio 3D

AI搭載の次世代3Dファッション物撮りアプリケーション。

## デプロイ手順 (Vercel)

1. GitHubにこのリポジトリをプッシュします。
2. [Vercel](https://vercel.com) にログインし、`New Project` からこのリポジトリを選択します。
3. `Environment Variables` に以下を設定します：
   - `API_KEY`: 取得したGemini APIキー
4. `Deploy` ボタンをクリックして完了です。

## 技術スタック
- React 19
- Three.js / React Three Fiber
- Tailwind CSS
- Gemini 3 Pro / Flash (AI Engine)
